Welcome Kernel Upgrader Project

This is simple bash script to update your kernel to the latest kernel version on www.kernel.org.

How To Use

$ git clone git@github.com:omegahanggara/kernel-upgrader.git
$ cd kernel-upgrader
$ bash src/kernel-upgrader.sh
